# Creepy Chocolate Collector

5×3, 20-line slot with cash-collection Free Spins featuring Collector Wilds and Chocolate Cash symbols.

## Core Features

**Grid:** 5 reels × 3 rows
**Lines:** 20 fixed paylines, pays left-to-right  
**RTP:** 95%
**Max Win:** 10,000× bet per bonus (hard cap)

## Symbols

**Low:** L1–L5 (card symbols)
**High:** H1–H5 (themed symbols)
**Wild (W):** Base game wild - substitutes all except scatter, no collection ability
**Collector Wild (CW):** Bonus-only wild that collects all Chocolate Cash values + multiplier ladder
**Jar Scatter (JAR):** Bonus trigger (appears base game only)
**Chocolate Cash (CC):** Coin symbols with cash values (2×, 5×, 10×, 15×, 20×, 25×, 50×, 2000× total bet)

## Base Game

Standard 20-line gameplay with regular wilds and scatter collection for bonus trigger.

## Bonus (Free Spins)

**Trigger:** 3/4/5 JAR → 10/15/20 free spins
**Collection:** Collector Wilds collect all Chocolate Cash values on the same spin
**Multiplier Ladder:** Every 4 Collector Wilds → +10 free spins and multiplier step (×1 → ×2 → ×3 → ×10)
**Multiplier Application:** Applied to collected Chocolate Cash only, not line wins
**Bonus Cap:** 10,000× total bet maximum payout
**No Scatters:** JAR scatters do not appear in bonus mode

## Key Mechanics

- Collector Wilds only appear in bonus mode
- Each Collector Wild collects ALL Chocolate Cash symbols on screen
- Multiplier applies to the total collected amount per spin
- After reaching ×10 multiplier, further sets of 4 CW still add +10 spins
- Bonus ends when spins run out OR 10,000× cap is reached
